# CareSync Project Rules

## Project Purpose
CareSync is a Health Informatics portfolio application focused on:
- Home-care patient logging
- Healthcare accessibility
- Mobile-first UX
- Frontline caregiver workflows

## Development Rules
- Mobile-first design.
- Accessibility-first architecture.
- Run lint, typecheck, and build before reporting completion.
- Do not introduce authentication unless requested.
- Do not create unnecessary backend complexity.
- Keep UI clean and healthcare-focused.
