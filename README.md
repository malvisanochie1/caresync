# CareSync

CareSync is a modern healthcare coordination platform designed to streamline communication and care management between **Personal Support Workers (PSWs)**, healthcare administrators, patients, and their families.

The platform enables healthcare providers to efficiently manage patient care, document visits, assign support workers, monitor progress, and improve collaboration across the care team. CareSync is being built with accessibility, scalability, and user experience at its core, ensuring the application remains usable for individuals with varying physical and cognitive needs.

> **Project Status:** 🚧 Active Development

---

## Overview

Managing patient care often involves multiple people working together, including healthcare providers, support workers, patients, and family members. Without a centralized system, tracking visits, documenting care, and monitoring patient progress can become difficult.

CareSync provides a centralized platform where healthcare teams can:

* Manage patients and care workers
* Record and review care logs
* Track patient visits
* Monitor care progress
* Improve communication between stakeholders
* Deliver a more accessible digital healthcare experience

### Accessibility First

CareSync is designed with accessibility as a core feature rather than an afterthought.

Users can personalize the interface based on their individual needs through built-in accessibility preferences.

Accessibility features include:

* Low Vision preset
* Dyslexia preset
* Senior mode
* Night Shift mode
* Adjustable font sizes
* Multiple readable font styles
* High contrast themes
* Background customization
* Text colour customization
* Motion reduction
* Enhanced keyboard focus indicators
* Adjustable spacing
* Persistent accessibility preferences across sessions

## Running Locally

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Visit:

```
http://localhost:3000
```
